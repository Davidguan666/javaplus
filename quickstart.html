Quick Start Guide

Get up and running with java+ in 5 minutes! This guide will walk you through creating your first java+ program.

1Install java+

# Clone the repository
git clone https://github.com/javaplus/javaplus.git
cd javaplus

# Run the installer
./install.sh

# Verify installation
jp --version

2Create Your First Program

Create a file called hello.jpp:

// hello.jpp
// My first java+ program

start() {
    print("Hello, World!");
    print("Welcome to java+ programming!");
}

3Run Your Program

jp run hello.jpp

You should see:

Hello, World!
Welcome to java+ programming!

4Create a Game Window

Create window.jpp:

// window.jpp
// Create a game window like pygame

import std.graphics;

start() {
    // Create window (like pygame.display.set_mode())
    let window = Window.create("My Game", 800, 600);
    
    // Game loop
    while (window.isOpen()) {
        // Handle events
        window.pollEvents();
        
        // Clear screen
        window.clear(Color.BLACK);
        
        // Draw a rectangle
        window.drawRect(100, 100, 200, 150, Color.ORANGE);
        
        // Show what we drew
        window.display();
    }
}

5Run the Game

jp run window.jpp

A window should open with an orange rectangle!

Next: Create a Project

Use the jp new command to create a complete project structure:

jp new MyAwesomeGame
cd MyAwesomeGame
jp run

This creates:

MyAwesomeGame/
├── jpp.json          # Project configuration
├── src/
│   └── main.jpp      # Main game file
├── assets/
│   ├── sprites/      # Images
│   ├── sounds/       # Audio
│   └── fonts/        # Fonts
└── build/            # Compiled output

Your First Real Game

Here's a complete Pong game:

// pong.jpp
import std.graphics;
import std.input;

// Game variables
let ballX = 400.0;
let ballY = 300.0;
let ballDX = 5.0;
let ballDY = 3.0;
let paddleY = 250.0;
let score = 0;

start() {
    let window = Window.create("Pong", 800, 600);
    window.setFramerateLimit(60);
    
    while (window.isOpen()) {
        window.pollEvents();
        
        // Exit on ESC
        if (Input.isKeyPressed(Key.ESCAPE)) {
            window.close();
        }
        
        // Move paddle
        if (Input.isKeyDown(Key.UP) && paddleY > 0) {
            paddleY = paddleY - 5;
        }
        if (Input.isKeyDown(Key.DOWN) && paddleY < 500) {
            paddleY = paddleY + 5;
        }
        
        // Update ball
        ballX = ballX + ballDX;
        ballY = ballY + ballDY;
        
        // Bounce off walls
        if (ballY <= 0 || ballY >= 580) {
            ballDY = -ballDY;
        }
        if (ballX >= 780) {
            ballDX = -ballDX;
        }
        
        // Paddle collision
        if (ballX <= 30 && ballY >= paddleY && ballY <= paddleY + 100) {
            ballDX = -ballDX;
            score = score + 1;
        }
        
        // Reset if missed
        if (ballX < 0) {
            ballX = 400;
            ballY = 300;
            score = 0;
        }
        
        // Draw everything
        window.clear(Color.BLACK);
        window.drawRect(10, paddleY, 20, 100, Color.WHITE);  // Paddle
        window.drawCircle(ballX, ballY, 10, Color.ORANGE);   // Ball
        window.display();
    }
}
Try it: Save the code above as pong.jpp and run with jp run pong.jpp
