Welcome to java+

java+ is a modern game development programming language designed to make game creation simple, intuitive, and fun.

Combining the best features of Java with modern syntax and game-focused capabilities, java+ provides everything you need to bring your game ideas to life.

🎮 Game-First Design

Built specifically for game development with entities, scenes, and game loops as first-class citizens.

⚡ Modern Syntax

Clean, readable code with type inference, lambda expressions, and modern control flow.

📦 Rich Standard Library

Built-in support for vectors, colors, math functions, and game development utilities.

Quick Example

// Hello World in java+
start() {
    print("Hello, Game World!");
}

// Simple game window
import std.graphics;

start() {
    // Create a window like pygame
    let window = Window.create("My Game", 800, 600);
    
    while (window.isOpen()) {
        window.clear(Color.BLACK);
        
        // Draw a rectangle
        window.drawRect(100, 100, 200, 150, Color.ORANGE);
        
        window.display();
    }
}
