Creating Windows

Creating a game window in java+ is simple and similar to pygame. This guide shows you how to create windows, handle the game loop, and manage the window lifecycle.

Basic Window Creation

The simplest way to create a window is using the Window class from the graphics library:

import std.graphics;

start() {
    // Create a window (like pygame.display.set_mode())
    let window = Window.create("My Game", 800, 600);
    
    // Main game loop
    while (window.isOpen()) {
        // Handle events
        window.pollEvents();
        
        // Clear the screen
        window.clear(Color.BLACK);
        
        // Draw your game here
        
        // Display what was drawn
        window.display();
    }
}
Like Pygame: If you're familiar with pygame, this is equivalent to: 
screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("My Game")

Window Configuration

You can configure various window properties:

import std.graphics;

start() {
    // Window configuration
    let config = WindowConfig {
        width: 1280,
        height: 720,
        title: "My Awesome Game",
        resizable: true,
        fullscreen: false,
        vsync: true,
        fps: 60
    };
    
    // Create window with config
    let window = Window.createWithConfig(config);
    
    // Or modify after creation
    window.setTitle("New Title");
    window.setSize(1024, 768);
    window.setFullscreen(false);
    window.setFramerateLimit(60);
}

Window Methods

Window.create(title, width, height)

Parameters:

  • title (string): Window title bar text
  • width (int): Window width in pixels
  • height (int): Window height in pixels

Returns: Window instance

window.isOpen()

Returns true if the window is currently open.

Use this in your game loop condition.

window.pollEvents()

Processes all pending window events (close, resize, focus, etc.).

Call this every frame in your game loop.

window.clear(color)

Clears the window with the specified color.

Parameters:

  • color (Color): Background color

window.display()

Displays everything that was drawn since the last clear.

Call this at the end of your drawing code.

Complete Example: Bouncing Ball

Here's a complete example showing a bouncing ball - similar to a pygame tutorial:

import std.graphics;
import std.math;

// Ball properties
let ballX = 400.0;
let ballY = 300.0;
let ballRadius = 20.0;
let velocityX = 5.0;
let velocityY = 3.0;

start() {
    // Create window like pygame
    let window = Window.create("Bouncing Ball", 800, 600);
    window.setFramerateLimit(60);
    
    // Game loop
    while (window.isOpen()) {
        // Handle events
        window.pollEvents();
        
        // Close window on escape key
        if (Input.isKeyPressed(Key.ESCAPE)) {
            window.close();
        }
        
        // Update ball position
        ballX = ballX + velocityX;
        ballY = ballY + velocityY;
        
        // Bounce off walls
        if (ballX - ballRadius <= 0 || ballX + ballRadius >= 800) {
            velocityX = -velocityX;
        }
        if (ballY - ballRadius <= 0 || ballY + ballRadius >= 600) {
            velocityY = -velocityY;
        }
        
        // Clear screen
        window.clear(Color.BLACK);
        
        // Draw ball
        window.drawCircle(ballX, ballY, ballRadius, Color.ORANGE);
        
        // Display
        window.display();
    }
}

Multiple Windows

java+ supports multiple windows (useful for tools or multi-monitor setups):

import std.graphics;

start() {
    // Create multiple windows
    let mainWindow = Window.create("Main Game", 1280, 720);
    let toolWindow = Window.create("Tools", 400, 600);
    
    while (mainWindow.isOpen()) {
        // Poll events for all windows
        mainWindow.pollEvents();
        toolWindow.pollEvents();
        
        // Update main window
        mainWindow.clear(Color.BLACK);
        // ... draw game ...
        mainWindow.display();
        
        // Update tool window
        toolWindow.clear(Color.DARK_GRAY);
        // ... draw tools ...
        toolWindow.display();
    }
}

Window Events

Handle window events for responsive applications:

import std.graphics;

start() {
    let window = Window.create("Event Demo", 800, 600);
    
    while (window.isOpen()) {
        // Poll events and handle them
        while (let event = window.pollEvent()) {
            match (event.type) {
                case Event.CLOSED:
                    window.close();
                case Event.RESIZED:
                    print("Window resized to: " + event.width + "x" + event.height);
                case Event.KEY_PRESSED:
                    if (event.key == Key.F11) {
                        window.toggleFullscreen();
                    }
                case Event.MOUSE_MOVED:
                    print("Mouse at: " + event.x + ", " + event.y);
            }
        }
        
        window.clear(Color.BLACK);
        window.display();
    }
}

Window Properties Reference

Property Type Description
width int Window width in pixels
height int Window height in pixels
title string Window title bar text
resizable bool Whether window can be resized
fullscreen bool Fullscreen mode
vsync bool Vertical sync enabled
fps int Target frames per second
Performance Tip: Always use window.setFramerateLimit() or vsync to prevent your game from using 100% CPU.
