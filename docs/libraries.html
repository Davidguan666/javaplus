Using Libraries

java+ has a powerful module system that allows you to import and use libraries. This guide covers the standard library, third-party libraries, and creating your own modules.

Importing Modules

Use the import keyword to include libraries in your code:

// Import the entire module
import std.io;
import std.math;
import std.graphics;

// Now use the imported functions
start() {
    print("Hello from std.io!");
    let result = Math.sqrt(16);
}

Standard Library Modules

Module Description Key Functions
std.io Input/Output operations print(), readLine()
std.math Mathematical functions sqrt(), sin(), cos(), pow(), random()
std.graphics Window and graphics Window, Color, draw functions
std.input Keyboard and mouse input Input.isKeyPressed(), Mouse.position()
std.time Time and timing functions Time.now(), Time.deltaTime()
std.audio Sound and music Sound.load(), Music.play()
std.json JSON parsing JSON.parse(), JSON.stringify()
std.file File operations File.read(), File.write()

Module Usage Examples

Math Module

import std.math;

start() {
    // Basic math
    print(Math.sqrt(16));        // 4.0
    print(Math.pow(2, 8));       // 256.0
    print(Math.abs(-5));         // 5
    
    // Trigonometry
    print(Math.sin(Math.PI / 2));  // 1.0
    print(Math.cos(0));             // 1.0
    
    // Random numbers
    print(Math.random());        // 0.0 to 1.0
    print(Math.randomInt(1, 6)); // Random int 1-6 (like dice)
    
    // Constants
    print(Math.PI);              // 3.14159...
    print(Math.E);               // 2.71828...
}

Time Module

import std.time;
import std.io;

start() {
    // Get current time
    let now = Time.now();
    print("Current timestamp: " + now);
    
    // Measure elapsed time
    let startTime = Time.now();
    
    // ... do some work ...
    
    let elapsed = Time.now() - startTime;
    print("Elapsed: " + elapsed + " seconds");
    
    // Sleep
    Time.sleep(1.0);  // Sleep for 1 second
    
    // Game timing
    while (true) {
        let deltaTime = Time.deltaTime();  // Time since last frame
        print("FPS: " + (1.0 / deltaTime));
    }
}

File Module

import std.file;
import std.io;

start() {
    // Read a file
    let content = File.read("data.txt");
    print(content);
    
    // Write to a file
    File.write("output.txt", "Hello, World!");
    
    // Check if file exists
    if (File.exists("savegame.dat")) {
        print("Save file found!");
    }
    
    // Read JSON
    let json = File.readJSON("config.json");
    print(json.playerName);
}

Creating Custom Libraries

You can create your own reusable libraries:

1. Create a Library File

Create lib/myutils.jpp:

// lib/myutils.jpp
// My custom utility library

// Clamp a value between min and max
func clamp(value, min, max) {
    if (value < min) return min;
    if (value > max) return max;
    return value;
}

// Linear interpolation
func lerp(start, end, t) {
    return start + (end - start) * t;
}

// Check if point is in rectangle
func pointInRect(px, py, rx, ry, rw, rh) {
    return px >= rx && px <= rx + rw &&
           py >= ry && py <= ry + rh;
}

// Export the module
export {
    clamp,
    lerp,
    pointInRect
};

2. Use Your Library

// main.jpp
import lib.myutils;

start() {
    // Use custom library functions
    let value = clamp(150, 0, 100);  // Returns 100
    let pos = lerp(0, 100, 0.5);     // Returns 50
    
    if (pointInRect(50, 50, 0, 0, 100, 100)) {
        print("Point is inside!");
    }
}

Library Search Path

java+ looks for libraries in this order:

  1. Current directory - ./lib/
  2. Project directory - Relative to your jpp.json
  3. Standard library - Built-in modules
  4. JAVAPLUS_PATH - Environment variable
// Set custom library path
export JAVAPLUS_PATH="/path/to/my/libraries:$JAVAPLUS_PATH"

Third-Party Libraries

Install third-party libraries using the package manager:

# Install a library
jp install physics2d

# Install specific version
jp install physics2d@1.2.0

# List installed libraries
jp list

# Update all libraries
jp update

# Remove a library
jp remove physics2d

Using Installed Libraries

// Import installed third-party library
import physics2d;

start() {
    let world = PhysicsWorld.create();
    world.setGravity(0, 9.8);
    
    let body = world.createBody(BodyType.DYNAMIC);
    body.setPosition(100, 100);
}

Module Structure

A complete library structure:

mylibrary/
├── jpp.json          # Library manifest
├── src/
│   ├── main.jpp      # Main module code
│   ├── utils.jpp     # Utility functions
│   └── types.jpp     # Type definitions
├── assets/
│   └── sprites/      # Default assets
└── README.md         # Documentation

Library Manifest (jpp.json)

{
  "name": "mylibrary",
  "version": "1.0.0",
  "description": "My awesome game library",
  "author": "Your Name",
  "main": "src/main.jpp",
  "dependencies": {
    "std.graphics": ">=1.0.0",
    "another-lib": "^2.0.0"
  },
  "keywords": ["game", "physics", "2d"]
}

Best Practices

Library Best Practices:
  • Keep libraries focused on one purpose
  • Use clear, descriptive function names
  • Document your public API
  • Version your libraries properly
  • Handle errors gracefully
